# Invite the Bot
Dank Memer can be played in the bot's DMs, or you can invite it to your server so you can play with friends.

There are several ways to invite the bot to a server:
- With the <a href="https://invite.dankmemer.lol/" target="_blank">official link</a>
- Through the <a href="https://discord.com/application-directory/270904126974590976" target="_blank">Discord app directory</a> 
- Through <a href="https://top.gg/bot/memes" target="_blank">top.gg</a> or <a href="https://discordbotlist.com/bots/dank-memer" target="_blank">discordbotlist</a>
- If you share a server with the bot, view its profile and click "Add to Server"
- The <a href="/Bot-features/Utility-and-Config-Commands/Utility-Commands#Invite" target="_blank">/invite</a>  command in the bot's DMs or in a server where the bot is.

## Setting up the Bot
If you need help getting the bot setup in a server, the two guides below will help!
<a href="https://dankmemer.lol/tutorial/set-up-dank-memer" target="_blank">Dank Memer Setup Tutorial</a>
<a href="https://support.discord.com/hc/en-us/articles/4644915651095-Command-Permissions" target="_blank">Discord's Guide to Command Permissions</a> 

If the guides above do not answer your questions, you can also ask for help in the <a href="https://discord.com/invite/dankmemerbot" target="_blank">Dank Memer Support Server</a>.

<br>

## Server Commands
There are several commands that can be used at the server level to help with the bot.
- <a href="/Bot-features/Utility-and-Config-Commands/Config-Commands#Audit" target="_blank">/audit</a> - To view what changes have been made to Dank Memer settings on the server.
- <a href="/Bot-features/Utility-and-Config-Commands/Config-Commands#Automeme" target="_blank">/automeme</a> - A premium server perk where you can automatically post memes.
- <a href="/Bot-features/Utility-and-Config-Commands/Config-Commands#Disableuse" target="_blank">/disableuse</a> - Command to disable the use of specific items in the server, in specific channels, or by certain roles or users.
- <a href="/About-Dank-Memer/Premium-users#PremiumCommands" target="_blank">/premium premiumserver add</a> - For premium users to make the server premium.
- <a href="/Bot-features/Utility-and-Config-Commands/Config-Commands#ServerSettings" target="_blank">/serversettings</a> - For basic server settings. It is recommended that users read the guide entirely to  better understand server settings and options.
- <a href="/Bot-features/Currency-Commands/Serverevents-and-Giveaways#Serverevents" target="_blank">/serverevents</a> - For server events and giveaways
