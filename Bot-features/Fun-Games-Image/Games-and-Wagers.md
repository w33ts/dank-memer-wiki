# Games and Wagers
Dank Memer has several games that you can play with others for fun with the `/game` command, there is also `/fight`, or you can add a bet on games with the `/wager` command. 

**Available Games:**
- [Connect 4](/Bot-features/Fun-Games-Image/Games-and-Wagers#Connect4)
- [Fight](/Bot-features/Fun-Games-Image/Games-and-Wagers#Fight)
- [Pet Fight](/Bot-features/Fun-Games-Image/Games-and-Wagers#Petfight)
- [Rock,Paper, Scissors (RPS)](/Bot-features/Fun-Games-Image/Games-and-Wagers#RPS)
- [Tic-Tac-Toe (TTT)](/Bot-features/Fun-Games-Image/Games-and-Wagers#TTT)
- [Wagers](/Bot-features/Fun-Games-Image/Games-and-Wagers#Wagers)


## Fight {: #Fight}
You can fight with others by either using the `/fight` command, either by `/fight create` or `/fight quick`. 

These commands also lets you wager coins and/or items against others by adding the "**quantity**" and "**for_quantity**" options in the command, allowing you to do coins vs items crossbets too.

The biggest difference is that only `/fight quick` counts towards `/fight ranking` and that `/fight create` gives you full control on how you want to customize your fight.



<br>
<center>
<img src="/bot-features/games/fight_create.png" alt="new /fight create embed">
</center>

`/fight quick` has the following game mode options:
- Normal
- No Defend
- Kick Only
- Punch Only
- Half Health
- Surprise
- Chaos

There is a detailed tutorial on how the new fighting system and rankings work and information about all of the modes on the website <a href="https://dankmemer.lol/tutorial/fighting" target="_blank">here</a>.

## Games {: #Games}

### <font color =b32d2d>Connect 4</font> {: #Connect4}
You can play a friendly game of Connect4 with others by using the `/game connect4` command. In order to win, a user needs to get 4 of their color tokens in a row horizontally, vertically, or diagonally. 

<br>
<center>
<img src="/bot-features/games/connect4.png" alt="Connect4 Game">
</center>

### <font color =b32d2d>Petfight</font> {: #Petfight}
Users who own a pet can petfight against each other by using the `/game petfight` command. Each pet has a special attack they can do. If either of your pets is sleeping or exhausted from another fight, you have to wait until it wakes up again before you can fight.

<br>
<center>
<img src="/bot-features/games/petfight.png" alt="Petfight game">
</center>


### <font color =b32d2d>Rock, Paper, Scissors (RPS)</font> {: #RPS}
Play a game of Rock, Paper, Scissors, often called RPS by users, with others using the `/game rps` command. Each user takes a turn picking their choice, and a winner is revealed.

**Winning Moves:**
- Rock > Scissors
- Paper > Rock
- Scissors > Paper

<br>
<center>
<img src="/bot-features/games/rps.png" alt="RPS Game">
</center>

### <font color =b32d2d>Tic-Tac-Toe (TTT)</font> {: #TTT}
You can play a fast game of Tic Tac Toe, often called TTT by users, by using the `/game tictactoe` command. Users take turns choosing a space until someone gets three of their symbols in a row horizontally, vertically, or diagonally. If the board fills up without a winner, the game ends in a draw.

<br>
<center>
<img src="/bot-features/games/tictactoe.png" alt="TTT Game">
</center>


## Wagers {: #Wagers}
If you want to bet item vs item or coins vs coins on any of the games above, you can do so using the `/wager` command. 

The wagered items go to the winner at the end of the fight, and if the fight ends in a draw, the users get their items back.

**Wagering items or coins works for the following games:**
- [Connect 4](/Bot-features/Fun-Games-Image/Games-and-Wagers#Connect4)
- [Fight](/Bot-features/Fun-Games-Image/Games-and-Wagers#Fight)
- [Pet Fight](/Bot-features/Fun-Games-Image/Games-and-Wagers#Petfight)
- [Rock, Paper, Scissors (RPS)](/Bot-features/Fun-Games-Image/Games-and-Wagers#RPS)
- [Tic-Tac-Toe (TTT)](/Bot-features/Fun-Games-Image/Games-and-Wagers#TTT)
