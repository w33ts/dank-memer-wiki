# Help Commands
The `/help` command is there to help you find more (detailed) information about certain aspects of the bot.

You can find more info about the following aspects

- [Categories](/Resources/help#HelpCategories) - A complete command list.
- [Commands](/Resources/help#HelpCommand) - Basic info about each command.
- [Start](/Resources/help#HelpStart) - The Dank Memer welcome message.
- [Table](/Resources/help#HelpTable) - The loot tables for various commands.
- [Tips](/Resources/help#HelpTip) - Browse useful tips.
- [Tutorials](/Resources/help#HelpTutorial) - Basic command tutorials.

## Help Categories {: #HelpCategories}
With the `/help categories` command, you can find back **all** of the commands for the bot listed per category. Each command has a short description to help you learn more about it.

<br>
<center>
<img src="/resources/help/help_categories.png" alt="help categories." width=357>
</center>

## Help Command {: #HelpCommand}
With `/help command` you can find more info for each command, such as:
- A small description of the command
- How to use the command and the possible subcommands
- The cooldowns for regular and <a href="/About-Dank-Memer/Premium-users" target="_blank">premium members</a>

<center>
  
![help_command_beg.png](/resources/help/help_command_beg.png)

</center>
 
  
## Help Start {: #HelpStart}
Using the `/help start` command will show you an ephemeral welcome message to the bot with some basic commands you can run to start your journey. 

At the bottom, you will find 4 buttons:
- <a href="/Bot-features/Currency-Commands/Farm" target="_blank">Farm</a>
- <a href="/Bot-features/Currency-Commands/Skins" target="_blank">Skins</a>
- <a href="/Bot-features/Fun-Games-Image/Games-and-Wagers" target="_blank">Games</a>
- <a href="/Bot-features/Currency-Commands/Grind-Commands" target="_blank">Grind</a>

Clicking each button will show you a small introduction to using the commands and provide links to some <a href="/Resources/Bot-tutorials" target="_blank">tutorials</a>.

<br>
<center>
<img src="/resources/help/help_start.png" alt="help tip." width=444>
</center>

## Help Table {: #HelpTable}
With the `/help table` command, you can check loot tables for a couple of <a href="/Bot-features/Currency-Commands/Grind-Commands" target="_blank">grind commands</a> to see what the possible rewards you can receive are.

Only the commands listed under `/help table` have a public loot table, and commands such as `/search` and `/crime` do **not** have a loot table. You can find a rough loot table of possible <a href="/Bot-features/Currency-Commands/Grind-Commands/Search" target="_blank">search locations and rewards here</a> and for <a href="/Bot-features/Currency-Commands/Grind-Commands/Crime" target="_blank">crime rewards here</a>.

<center>

  ![help_table_new.png](/resources/help/help_table_new.png)
  
  </center>
  
## Help Tip {: #HelpTip}
The bot has a lot of features, so Dank Memer offers tips for users to help them learn about commands and different parts of the bot. The `/help tip` command allows you to browse through all of the available tips for more information at any time.

<br>
<center>
<img src="/bot-features/settings/show_another_tip.png" alt="show another tip." width=447>
</center>

If you like tips, you can enable them to appear in ephemeral messages while you are <a href="/Bot-features/Currency-Commands/Grind-Commands" target="_blank">grinding</a>/using the bot. Ephemeral messages are messages only you can see, so they won't bother other users or flood the channel.

If you wish to enable/disable these tips, you can use the <a href="/Bot-features/Utility-and-Config-Commands/Config-Commands#Settings" target="_blank">/settings</a> command and look for the "Ephemeral Notifications" option in the dropdown menu. 

<br>
<center>
<img src="/bot-features/settings/ephemeral_notifs.png" alt="ephemeral notifications." width=447>
</center>

Click on the Customize button to view what types of notifications are available. Here you can enable/disable the tips shown by the bot, as well as notifications for things like double XP days, changelogs, quests, and alerts.

<br>
<center>
<img src="/bot-features/settings/tips_new.png" alt="tips." width=447>
</center>



## Help Tutorial {: #HelpTutorial}
With the `/help tutorial` command, you can check back on the small tutorials that are shown when you first run a command. These tutorials give basic information about what the command does and how to use it. If there is a longer tutorial available, these will also be linked in the description.

Not every command has a tutorial, but most of the basic currency commands do.

<br>
<center>
  <img src="/resources/help/help_tutorial_new.png" alt="Help tutorial command." Height=582>
  </center>
 

<br>
