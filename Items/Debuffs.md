# Debuff Items
Debuff items apply a negative effect to a user. These are all consumed on use, and both you and the user it is used on must be out of passive mode. These items can do things like give negative luck, stop someone from running commands, or prevent someone from robbing or using items on you. You can see the current market value, net value, and more about these items on the <a href="https://dankmemer.lol/items" target="_blank">Dank Memer Website</a>.
